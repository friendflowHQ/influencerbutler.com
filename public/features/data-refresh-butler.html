Without this

Stale commission rates. Daily Deals surfaces dead campaigns. You click “refresh” in five different butlers every morning.

With Butler

One schedule refreshes both catalogs in the background. Every downstream butler reads the latest data without you lifting a finger.

How it saves you time, money, and headaches

  • Never miss a commission rate change that would have boosted your earnings.
  • Daily Deals always runs on fresh data — no more posting deals on expired campaigns.
  • Split cadences so the slow catalog (CC campaigns) doesn’t slow down the fast one (SPCC rates).

What’s in the box

  • One-click cadence pills: 1h, 3h, 6h, 12h, 24h — or a custom time-of-day
  • Advanced split mode: keep CC on a slow daily cadence, run SPCC every hour
  • Coupled mode auto-refreshes 1 hour before each scheduled Daily Deals search
  • Live status cards for each catalog: last refresh, next refresh, error messages
  • Manual Run Now buttons for each catalog and a combined refresh
  • Searchable, sortable SPCC deals table with EPC, deal %, budget, rating, and ASIN deep-link
